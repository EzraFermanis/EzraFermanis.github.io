In the Javascript Language, describe all the use cases for each of:

Parenthesis ()

The parenthesis in Javascript is required when declaring, defining or using functions. The Parenthesis are required whether or not you use parameters or 'arguments' in a function. The parenthesis when calling or invoking a function are used to enclose the arguments you put into the function.

Brackets []

Brackets in Javascript are representative of the object type 'array' and are used to enclose 1 or more values. Arrays can hold numbers, strings and even functions. You can create an array by typing the name of the array and then the array of values you want that array to hold:

e.g: var simpleArray = ["I", "am", "an", "array"];

The brackets can also be used to select a particular value from and array by calling in by it's index. For example you could find out what the first value in the array we called is (note: array indexes start at 0) by typing simpleArray[0] and "I" will be returned

Braces {}

The braces contains a block of code or instructions. For example if a function says if certain conditions are met, then { do this code }.

Single Quotes ' '

Single Quotes encapsulate strings. Similiar to a double quote, a single quote is the representative symbol of the data type 'strings'.

Double Quotes " "

Double Quotes are pretty much the same as single quotes. You can use them both interchangably, however, if you start a string with a single quote you must end it with a single quote, and vice versa.
